# Apexplanet Task 5

## Overview
This project contains multiple web applications demonstrating:
- Form Validation
- API Integration
- Local Storage Management
- Responsive Design

## Features
- ✅ Input Validation (Name field)
- ✅ API Integration with JSONPlaceholder
- ✅ CRUD Operations with Local Storage
- ✅ Interactive UI with CSS styling
- ✅ Error Handling
- ✅ Responsive Design

## Tech Stack
- **HTML5** - Markup structure
- **CSS3** - Styling and responsive design
- **JavaScript (ES6+)** - Async/Await, Fetch API, DOM manipulation

## File Structure
```
├── index.html      # Main HTML file
├── script.js       # JavaScript functionality
├── styles.css      # CSS styling
└── README.md       # Documentation
```

## Installation & Usage

1. **Clone or download the repository**
   ```bash
   git clone https://github.com/FarhanSyed14/Apexplanettask5.git
   cd Apexplanettask5
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a live server extension in VS Code

## Features Breakdown

### 1. Form Validation
- Validates that the name field is not empty
- Displays "Name Required" or "Valid" message

### 2. API Integration
- Fetches posts from JSONPlaceholder API
- Displays first 5 posts
- Includes error handling for failed requests

### 3. Local Storage
- Save data to browser storage
- Retrieve stored data
- Clear stored data

## APIs Used
- **JSONPlaceholder** (https://jsonplaceholder.typicode.com/) - Test API for posts
